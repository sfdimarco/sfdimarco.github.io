Why .geoi matters for AI vision
The Problem

Vision AI tokenizes images as uniform 16×16 pixel grids. A plain wall costs the same tokens as a complex painting. Token count scales quadratically with resolution — and you pay per token.

GPT-4o    1080p image → ~765 tokens
Claude     1080p image → ~1,300 tokens
1M images/mo → $1,900–$3,900
The Quadtree Insight

A quadtree is a spatial hierarchy — dense where information is dense, sparse where it's uniform. 60–80% of a typical image is low-variance (sky, walls, backgrounds). Those regions become single leaf nodes. The important detail gets full depth.

SparseVLM (ICML 2025)  54% fewer tokens
FastV                 45% fewer tokens
97% accuracy retained ✓
The Gap .geoi Fills

All token-compression research happens inside training pipelines. There is no production image transport format that encodes spatial hierarchy for vision APIs. JPEG doesn't. PNG doesn't. WebP doesn't.

.geoi is a proposal for what that format looks like. Every prefix of the bitstream is a valid image — progressive by design, hierarchical by nature.
