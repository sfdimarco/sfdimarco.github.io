## About the Site

A dark-mode-first portfolio built in React + Vite. The design system is grounded in **grapheme-color synesthesia** — a neurological condition in which digits involuntarily appear as specific colors. Every color decision in this site is drawn from that perceptual map:

| Digit | Color |
|-------|-------|
| 0 | Cyan |
| 1 | Red |
| 2 | Blue |
| 3 | Yellow |
| 4 | Green |
| 5 | Orange |
| 6 | Purple |
| 7 | Pink |
| 8 | Near-white |
| 9 | Near-black |

This is called the **SYN system**. It runs as a constant across all projects.

## Pages

| Route | Content |
|-------|---------|
| `/` | Landing — animated SynGrid background, breathing name, portal cards |
| `/work` | Portfolio and archive |
| `/lab` | Tools in progress |
| `/about` | Full bio, synesthesia explainer, contact |

The landing page features a **BreathingGrid** — a canvas animation where digits from the SYN palette pulse, drift, and mutate in a ripple wave across the screen. It's not decorative. It's the site explaining what it is.

## Design Philosophy

> "I have synesthesia — numbers are colours, sounds are shapes. My UI decisions aren't aesthetic preferences. They're perceptual data."

This site is not built around a design system chosen from a mood board. It's built around a nervous system. Synesthetic perception is multi-channel processing — navigating token spaces, color gradients, and information density the way most people navigate physical rooms: by feel, shape, and color.
